## What is this?

AK-OS is a fully interactive, OS-themed portfolio that runs in your browser. It boots up like a real operating system ‚ÄĒ complete with a BIOS sequence, desktop environment, draggable windows, a working terminal, games, and a Clippy easter egg. Every window is a different section of the portfolio.

No templates. No UI libraries. Built entirely with Next.js 15, TypeScript, and raw CSS.

---

## Features

### OS Shell
- **BIOS Boot Sequence** ‚ÄĒ animated POST diagnostics with dynamic progress bar tied to boot lines
- **Welcome Screen** ‚ÄĒ glitch-animated name, CTA buttons, bootUp animation
- **Desktop Environment** ‚ÄĒ drag, minimize, maximize, close windows
- **Window Manager** ‚ÄĒ full z-index stacking, focus management, spawn positioning
- **Taskbar** ‚ÄĒ START menu, open window labels, live clock
- **Desktop Icons** ‚ÄĒ 2-column grid, hover states, double-click to open

### Portfolio Windows
- **About** ‚ÄĒ personal intro and background
- **Resume** ‚ÄĒ embedded PDF viewer with download
- **Projects** ‚ÄĒ case study grid (real + placeholder projects)
- **Experience** ‚ÄĒ system log aesthetic timeline
- **Tech Stack** ‚ÄĒ categorized skill tags
- **Skills Radar** ‚ÄĒ pure SVG radar chart
- **Career Timeline** ‚ÄĒ visual milestone tracker
- **Achievements** ‚ÄĒ trophy-style highlights
- **Certifications** ‚ÄĒ credential log

### Interactive Tools
- **Terminal** ‚ÄĒ 10+ commands (`help`, `whoami`, `ls`, `cat`, `neofetch`, and more)
- **File Manager** ‚ÄĒ navigable folder tree
- **Code Playground** ‚ÄĒ live JavaScript execution environment
- **Display Settings** ‚ÄĒ 9 animated wallpapers √ó 4 color themes √ó boot toggle
- **Contact** ‚ÄĒ mailto-wired contact badge + start menu entry

### Games & Easter Eggs
- **Snake** ‚ÄĒ classic snake game
- **Tetris** ‚ÄĒ fully playable Tetris
- **Clippy** ‚ÄĒ appears after 8s, cycles through 8 witty tips, auto-rotates every 6.5s

### Visual System
- **CRT Overlay** ‚ÄĒ dual-layer phosphor flicker + moving scanline beam
- **Phosphor Glow** ‚ÄĒ text-shadow glow system across UI elements
- **bootUp Animation** ‚ÄĒ CRT power-on effect on every window open
- **Glitch Effect** ‚ÄĒ CSS clip-rect glitch on welcome screen name
- **9 Wallpapers** ‚ÄĒ Matrix Rain, Starfield, Retro Grid, Pixel Clouds, Cyber Rain, Binary, Geometry, Solid Color, None
- **4 Themes** ‚ÄĒ Matrix (default), Green Phosphor, Amber, White
- **Mobile Gate** ‚ÄĒ SYSTEM ERROR screen for viewports < 768px

## Local Development

```bash
git clone https://github.com/AdityaKuchhal/ak-os.git
cd ak-os
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Environment Variables

```bash
RESEND_API_KEY=re_xxxxxxxxxxxx   # For contact form email
```

## Terminal Commands

Once inside the OS, open the Terminal and try:

| Command | Description |
|---|---|
| `help` | List all available commands |
| `whoami` | Display owner info |
| `ls` | List all applications |
| `neofetch` | System info in neofetch style |
| `cat resume` | Open resume window |
| `open <app>` | Open any application by name |
| `clear` | Clear terminal output |
| `matrix` | Toggle Matrix rain wallpaper |
